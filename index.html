Material

Los materiales son muy importantes cuando queremos crear los cubrelechos que haya requerido el usuario

Como Funciona

La fases para crear un cubrelecho, depende de la informacion que de el usuario, el usuario inspecciona el tipo de tela, pide un hilo especifico.

Proceso de creacion

se recolecta la informacion que dio el usuario para poder hacer la comunicaion a empresas exteriores y pedir los productos que se necesitan, cuando llega el tipo de tela lo primero que se hace es cortarla, lo pasan a la alcolchadora que eso es lo que hace para crear la forma del cubrelecho

Cuantos Productos se hacen

Al dia se estan haciendo 100 tipos de cubrelechos ya que hay varias sedes que ayudan a los usuarios a elegir nuestros pruductos

Tiempo De entraga

Depende del tipo de tela que pida el ususario, como la quiere, el tamaño y entre otras cosas los mas optimo para la entrega sea una semana aproximadamente.

Tipos De Productos

La tela de diferentes texturas,El Hilo, La pluma, La Hilaza, El Algodon, cubrelechos, Cobijas, Almohadas, etc.
